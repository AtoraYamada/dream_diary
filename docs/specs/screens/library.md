# 書斎（静寂の書斎 / メインハブ）

アプリのハブとなる画面。「ポイント＆クリック」形式。

---

## 画面構成

| 要素 | 位置 | クリック先 |
|------|------|-----------|
| 巻物 | 手前（机上） | 夢作成画面 |
| 本棚 | 正面（奥の壁） | 夢一覧画面 |
| 窓 | 左側 | 夢の氾濫演出 |
| 鏡 | 右隅 | ログアウト（目覚めの儀式） |

### 本棚の蔵書画像

夢日記の冊数によって本棚の見た目が変化。アセットリスト参照。

| 冊数 | アセット |
|------|----------|
| 0冊 | img_bookshelf_empty.png |
| 少量 | img_bookshelf_small.png（上段のみ） |
| 中量 | img_bookshelf_medium.png（中段まで） |
| 多量 | img_bookshelf_large.png（下段まで） |

---

## UI動作

### ホバー効果

クリック可能なオブジェクトはホバー時に発光。

| 要素 | ホバー動作 |
|------|-----------|
| 巻物 | 発光 |
| 本棚 | 発光 |
| 窓 | 七色に光りながら楕円が収縮 |
| 鏡 | 黄色く白く光りながら楕円が収縮 |

### クリック動作

| 要素 | 動作 |
|------|------|
| 巻物 | 瞬き → 作成画面へ遷移 |
| 本棚 | 瞬き → 一覧画面へ遷移 |
| 窓 | 夢の氾濫演出開始 |
| 鏡 | 目覚めの儀式（ログアウト） |

## API

### ログアウト

**エンドポイント**: `DELETE /users/sign_out`

**レスポンス（成功: 200）**:
```json
{
  "message": "Logged out successfully."
}
```
