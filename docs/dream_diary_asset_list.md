#### 1.2.5 本（詳細 - 見開き表示）

**実装方針**: div + CSS ベース（テキスト処理最適化）

**アセット**: なし（ページフレームは CSS で描画）

**実装構造**:
```html
<div class="book-open-container">
  <!-- 左ページ -->
  <div class="page left-page">
    <div class="page-frame">
      <div class="page-content left-page-content">
        <!-- ページ1のテキスト -->
      </div>
    </div>
  </div>

  <!-- 右ページ -->
  <div class="page right-page">
    <div class="page-frame">
      <div class="page-content right-page-content">
        <!-- ページ2のテキスト -->
      </div>
    </div>
  </div>
</div>

<div class="page-number">1/10</div>
```

**CSS**:
- ページフレーム: CSS で左右のページ形状と綴じ目の影（`linear-gradient`）を描画
- 背景色: 固定の羊皮紙色（感情彩色は本の背表紙のみに適用）
- めくり演出: `transform: rotateY()` で 3D 回転アニメーション

**利点**:
- テキストが実際の HTML → コピー・検索可能
- ページ分割処理が簡単（JavaScript で固定文字数分割）
- 3D めくり演出に対応
- 画像ファイル制作不要
