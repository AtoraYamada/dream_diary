# 夢詳細画面（記憶の断片）

見開きの本で夢日記を閲覧。背景は一覧画面（ぼかし）。

---

## 画面構成

| 要素 | 説明 |
|------|------|
| 背景 | 一覧画面（ぼかし） |
| 見開きフレーム | 本の枠（感情彩色で色分け） |
| 左ページ / 右ページ | フレーム上にhtml要素として配置 |
| 修正用の羽ペンとペーパーナイフ | 編集トリガー（砂時計上方） |
| アンティークな砂時計 | 削除ボタン（忘却の儀式） |

---

## ページ構成

タイトル・夢を見た日・本文の順でページング。感情彩色は本の色で視覚的に表現。

| ページ | 内容 |
|--------|------|
| 1ページ目（左） | タイトル、夢を見た日、本文冒頭 |
| 2ページ目（右） | 本文続き |
| 3ページ目以降 | 本文続き（ページめくりで表示） |

左右両ページに本文が乗る。

---

## UI動作

### ページめくり

| 操作 | 動作 |
|------|------|
| 右ページ端クリック | 次ページへ（rotateY 800ms） |
| 左ページ端クリック | 前ページへ（rotateY 800ms） |

### 編集へ遷移

| 操作 | 動作 |
|------|------|
| 羽ペン+ナイフクリック | 本閉じ → 瞬き → 編集画面 |

### 削除（忘却の儀式）

| 操作 | 動作 |
|------|------|
| 砂時計クリック | 忘却の儀式開始 |

### 閉じる

| 操作 | 動作 |
|------|------|
| 本の外側クリック | 本閉じ → 一覧画面へ |

---

## エラーハンドリング

エラーメッセージは `config/locales/ja.yml` 参照。

| 状況 | 対応 |
|------|------|
| 取得失敗（404） | 一覧画面へ戻る + エラーメッセージ |
| 削除失敗 | sfx_glitch_dissonance.wav + エラーメッセージ表示 |

---

## API

### 夢詳細取得

**エンドポイント**: `GET /api/v1/dreams/:id`

**レスポンス（成功: 200）**:
```json
{
  "id": 1,
  "title": "夢のタイトル",
  "content": "夢の本文...",
  "emotion_color": "peace",
  "lucid_dream_flag": false,
  "dreamed_at": "2025-01-15",
  "tags": [
    { "id": 1, "name": "田中", "category": "person" }
  ]
}
```

### 夢削除

**エンドポイント**: `DELETE /api/v1/dreams/:id`

**レスポンス（成功: 204）**: No Content
